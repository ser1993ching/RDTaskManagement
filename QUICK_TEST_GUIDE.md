# 快速测试指南

## 访问系统
- **地址**: http://localhost:3002/
- **账户**: admin / admin 或 张组长 / 123

## 5分钟快速测试

### 1. 删除分类测试 (1分钟)
1. 登录 → 设置 → 任务分类管理
2. 找到"市场配合"卡片
3. 悬停在一个分类上（如"标书"）
4. 点击红色删除按钮（🗑️）
5. 确认删除

**结果**: ✅ 分类消失，显示成功提示

### 2. 拖拽排序测试 (2分钟)
1. 仍在"市场配合"卡片
2. 点击☰图标并拖拽到其他位置
3. 释放鼠标

**结果**: ✅ 分类顺序调整，显示成功提示

### 3. 添加分类测试 (1分钟)
1. 在"新增分类"输入框输入"测试分类"
2. 点击"添加"按钮

**结果**: ✅ 分类出现在列表末尾

### 4. 编辑分类测试 (1分钟)
1. 悬停在"测试分类"上
2. 点击蓝色编辑按钮（✏️）
3. 修改为"新测试分类"
4. 点击"保存"

**结果**: ✅ 名称更新，显示成功提示

### 5. 同步验证 (1分钟)
1. 进入"任务管理"
2. 选择"市场配合"任务类别
3. 查看分类筛选下拉框

**结果**: ✅ 顺序与设置界面一致

## 如果功能不工作

### 检查1: 浏览器控制台
1. 按F12打开开发者工具
2. 查看Console选项卡
3. 是否有红色错误信息？

### 检查2: 网络请求
1. 在Network选项卡中
2. 刷新页面
3. 是否有失败的请求？

### 检查3: localStorage
1. 在Console中输入：`localStorage.getItem('rd_task_categories')`
2. 是否返回JSON数据？
