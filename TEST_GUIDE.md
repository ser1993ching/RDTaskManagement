# 任务分类管理功能测试指南

## 测试环境

- **访问地址**: http://localhost:3002/
- **默认账户**: admin / admin 或 张组长 / 123

## 测试步骤

### 1. 登录系统
1. 打开浏览器访问 http://localhost:3002/
2. 使用以下任一账户登录：
   - 用户名: `admin` 密码: `admin` (管理员)
   - 用户名: `张组长` 密码: `123` (班组长)

### 2. 进入设置页面
1. 登录后在左侧导航栏点击"设置"
2. 进入设置页面后会看到多个标签页

### 3. 测试任务分类管理功能

#### 3.1 访问任务分类管理
1. 点击"任务分类管理"标签页
2. 页面会显示任务类别选择器

#### 3.2 选择任务类别
1. 在"选择任务类别"下拉框中选择一个类别，例如：
   - 市场配合 (MARKET)
   - 项目执行 (EXECUTION)
   - 产品研发 (PRODUCT_DEV)
2. 选择后下方会显示该类别下的所有二级分类

#### 3.3 查看分类列表
1. 观察分类列表的显示
2. 检查是否显示了正确的分类数量
3. 如果某个类别没有分类，会显示"该任务类别下暂无分类"的提示

#### 3.4 添加新分类
1. 在"新增分类"输入框中输入分类名称，例如："测试分类"
2. 点击"添加"按钮
3. 验证：
   - 新分类是否出现在列表中
   - 分类计数是否更新
   - 是否显示成功提示

#### 3.5 编辑分类名称
1. 找到任意一个分类项
2. 点击分类项右上角的编辑按钮（✏️图标）
3. 验证：
   - 分类项是否切换到编辑模式
   - 输入框是否自动聚焦
   - 是否显示保存和取消按钮
4. 修改分类名称，例如将"标书"改为"标书制作"
5. 点击"保存"按钮
6. 验证：
   - 是否显示"任务分类更新成功"提示
   - 分类名称是否更新为新值
   - 是否自动退出编辑模式

#### 3.6 取消编辑
1. 点击任意分类的编辑按钮
2. 修改输入框中的内容
3. 点击"取消"按钮
4. 验证：
   - 分类名称是否保持原值
   - 是否自动退出编辑模式

#### 3.7 删除分类
1. 找到任意一个分类项
2. 点击删除按钮（🗑️图标）
3. 在弹出的确认对话框中点击"确定"
4. 验证：
   - 是否显示"任务分类删除成功"提示
   - 分类是否从列表中移除
   - 分类计数是否更新

#### 3.8 取消删除
1. 点击任意分类的删除按钮
2. 在确认对话框中点击"取消"或关闭对话框
3. 验证：分类是否保留在列表中

### 4. 权限测试（使用不同角色）

#### 4.1 管理员/班组长权限
1. 使用admin或张组长账户登录
2. 进入任务分类管理
3. 验证：
   - ✅ 可以查看所有分类
   - ✅ 可以添加新分类
   - ✅ 可以编辑分类名称
   - ✅ 可以删除分类
   - ✅ 显示所有操作按钮

#### 4.2 组员权限（如果存在组员账户）
1. 使用组员账户登录（如果有的话）
2. 进入任务分类管理
3. 验证：
   - ✅ 可以查看分类列表
   - ❌ 不能添加分类（新增分类区域隐藏）
   - ❌ 不能编辑分类（编辑按钮隐藏）
   - ❌ 不能删除分类（删除按钮隐藏）
   - 显示权限警告："只有管理员和班组长可以管理任务分类"

### 5. 边界情况测试

#### 5.1 空名称验证
1. 点击编辑按钮进入编辑模式
2. 清空输入框
3. 点击保存
4. 验证：是否显示"分类名称不能为空"的错误提示

#### 5.2 快速操作测试
1. 快速连续点击同一个分类的编辑按钮
2. 快速连续点击多个分类的编辑按钮
3. 验证：是否正常工作，没有异常

#### 5.3 编辑时切换类别
1. 进入某个类别的分类编辑模式
2. 不保存也不取消
3. 切换到其他任务类别
4. 验证：是否正常切换，编辑状态是否重置

### 6. 响应式测试

#### 6.1 桌面端（1920x1080）
1. 调整浏览器窗口大小
2. 验证：分类卡片是否为3列布局

#### 6.2 平板端（768x1024）
1. 调整浏览器窗口大小或使用平板设备
2. 验证：分类卡片是否为2列布局

#### 6.3 手机端（375x667）
1. 调整浏览器窗口大小或使用手机设备
2. 验证：分类卡片是否为1列布局

### 常见问题排查

#### 问题1：无法看到"任务分类管理"标签页
- **原因**: 用户权限不足
- **解决**: 使用管理员或班组长账户登录

#### 问题2：编辑后数据没有保存
- **原因**: 点击了取消按钮或输入了空名称
- **解决**: 确保点击保存按钮且输入非空名称

#### 问题3：分类列表为空
- **原因**: 选择了没有配置分类的任务类别
- **解决**: 选择其他任务类别或先添加分类
