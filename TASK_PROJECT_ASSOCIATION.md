# 任务与项目关联优化说明

## 优化概述

本次优化实现了任务类型与项目类别的严格关联，确保数据一致性和系统规范性。

## 关联关系映射

| 任务类型 | 项目类别 | 说明 |
|---------|---------|------|
| 市场配合 (MARKET) | 市场配合项目 | 市场投标、技术方案、复询等任务 |
| 项目执行 (EXECUTION) | 项目执行项目 | 设计院提资、CT配合、图纸会签等任务 |
| 产品研发 (PRODUCT_DEV) | 科研项目 | 产品研发相关任务 |
| 科研任务 (RESEARCH) | 科研项目 | 科研项目相关任务 |
| 改造服务 (RENOVATION) | 改造项目 | 改造服务相关任务 |
| 其他任务 (OTHER) | 其他项目 | 通用任务 |

## 主要功能改进

### 1. 项目过滤
- 创建/编辑任务时，关联项目下拉列表仅显示匹配的项目类别
- 标签显示当前筛选的项目类型

### 2. 自动项目创建
- 新建项目时，根据当前任务类型自动设置正确的项目类别
- 确认对话框显示将要创建的项目类别

### 3. 自动匹配优化
- 根据任务名称自动匹配项目时，仅从匹配类别中查找
- 提高匹配的准确性

### 4. 编辑验证
- 编辑任务时，如果项目与任务类型不匹配，显示警告提示
- 提示用户调整任务类型或更换项目

### 5. 保存验证
- 保存任务时严格验证项目与任务类型匹配
- 不匹配的数据无法保存，确保数据一致性

## 使用说明

### 创建任务
1. 进入任务管理，选择任务类型
2. 点击"创建任务"
3. 在"关联项目"字段中，只能看到匹配类别的项目
4. 如果没有匹配项目，可以点击"+ 新建项目"创建
5. 填写其他信息并保存

### 编辑任务
1. 双击任务进入编辑模式
2. 如果当前项目与任务类型不匹配，会显示警告提示
3. 可以更换为匹配的项目，或调整任务类型
4. 保存时会再次验证匹配性

### 项目创建
1. 在任务创建页面，点击"关联项目"下的"+ 新建项目"
2. 输入项目名称
3. 确认对话框会显示将要创建的项目类别（根据当前任务类型自动确定）
4. 确认创建

## 技术实现

### 核心函数
- `getProjectCategoryForTaskClass()`: 获取任务类型对应的项目类别
- `getProjectsForTaskClass()`: 根据任务类型过滤项目列表

### 验证点
1. 打开编辑模态框时：检查现有项目的匹配性
2. 自动匹配项目时：从匹配类别中查找
3. 保存任务时：最终验证项目与任务类型匹配

## 注意事项

1. **数据兼容性**: 现有不匹配的数据不会影响系统运行，但编辑时会提示
2. **项目类别调整**: 如需调整项目类别，请在项目管理中修改
3. **任务类型切换**: 切换任务类型时，关联项目会自动更新为匹配类别

## 测试建议

1. 创建不同类型的任务，验证项目过滤是否正确
2. 尝试编辑现有任务，检查警告提示是否正常
3. 测试新建项目功能，验证类别自动设置
4. 尝试保存不匹配的任务，验证保存拦截
